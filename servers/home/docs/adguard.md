# AdGuard Home

Network-wide ad blocking DNS server.

## Features

- DNS-level ad blocking
- Tracking protection
- Parental controls
- Custom filtering rules
- Statistics dashboard

## Setup

**Initial configuration**: Access `https://dns.${DOMAIN}:3000` on first run.

**Configure devices**: Set DNS to server's IP address.

## Access

Web UI: `https://dns.${DOMAIN}`
