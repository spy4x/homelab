# Transmission

BitTorrent client for downloading torrents.

## Features

- Web-based interface
- RSS feed support
- Scheduling
- Blocklist support
- Remote control apps

## Access

Web UI: `https://torrents.${DOMAIN}`

## Mobile Apps

- [Transmission Remote GUI](https://github.com/transmission-remote-gui/transgui)
- [Transdroid](http://www.transdroid.org/) (Android)
