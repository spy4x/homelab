# MeTube

YouTube downloader web interface.

## Features

- Download YouTube videos/playlists
- Audio-only downloads
- Format selection
- Queue management
- Download history

## Access

Web UI: `https://metube.${DOMAIN}`

## Usage

Paste YouTube URL and click download. Files saved to configured output directory.
